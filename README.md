# Super Mario SpongeBob

Game sederhana bergaya platformer dengan tema SpongeBob. Pemain mengendalikan karakter SpongeBob, melompat, menghindari rintangan diam dan bergerak, serta menembakkan gelembung.

## Fitur

- 2 level dengan rintangan diam dan musuh bergerak
- 3 nyawa di setiap level
- Amunisi gelembung awal hanya 3 kali tembakan
- Kontainer hadiah untuk menambah amunisi
- Reset level otomatis saat nyawa habis

## Kontrol

- `Arrow Left` / `Arrow Right`: bergerak kiri/kanan
- `Arrow Up` atau `W`: lompat
- `Space`: tembak gelembung

## Cara bermain

1. Buka `index.html` di browser.
2. Gunakan tombol keyboard untuk bergerak dan lompat.
3. Tembak rintangan yang bisa dihancurkan dengan gelembung.
4. Ambil kotak hadiah hijau untuk menambah amunisi.
5. Jika nyawa habis, level akan diulang.

## Cara menjalankan

- Buka `index.html` langsung di browser.
- Atau jalankan server lokal sederhana jika diperlukan.
